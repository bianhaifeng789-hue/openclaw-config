# Growth Team Agents

这是一套面向 App 研发公司增长运营场景的 4 Agent 最小闭环：

1. **ua-lead** - 投放决策官
2. **monetization** - 变现优化官
3. **ltv-analyst** - LTV留存分析官
4. **creative-strategist** - 素材策略官

## 推荐协作链路

### 链路A：日报驱动
日报/CSV → ltv-analyst / monetization / creative-strategist → ua-lead 汇总决策

### 链路B：周会驱动
周报 → 3个分析Agent输出专项建议 → ua-lead 形成下周预算与动作

## 输入示例
- IAA日报 CSV
- 广告平台分国家 ROI 报表
- 留存/LTV 数据
- 素材表现报表（CTR/CVR/ROI）

## 输出示例
- 加量/降量/止损决策
- 变现配置建议
- 留存与回收周期判断
- 素材测试池

## 当前定位
这套 Agent 先解决“增长运营中台”问题，不替代研发多 Agent 系统。
