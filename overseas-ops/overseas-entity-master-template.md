# 海外主体总表（模板）

> 用途：统一管理海外公司主体、执照、联系人、税务、可投放/可注册平台、资料缺失项。

## 字段说明

| 字段 | 说明 |
|---|---|
| 主体ID | 内部唯一标识，便于映射 |
| 主体名称（中文） | 公司中文名 |
| 主体名称（英文） | 公司英文名/注册名 |
| 注册国家/地区 | 如 HK / SG / US / UAE |
| 主体类型 | 公司 / 个体 / 壳主体 / 代理主体 |
| 注册号 | 公司注册号/商业登记号 |
| 成立日期 | 公司成立时间 |
| 营业状态 | 正常 / 待补资料 / 停用 |
| 执照文件 | 执照/注册证书位置 |
| 地址 | 注册地址 |
| 联系人 | 对接人 |
| 联系方式 | 邮箱/电话 |
| 法人/董事 | 主体负责人 |
| 税务信息 | VAT/EIN/税号等 |
| 收款信息 | 银行账户/收款通道 |
| 可用平台 | 可开户/可投放的平台 |
| 适用产品 | 对应产品或业务线 |
| 风险备注 | 风险、限制、历史问题 |
| 缺失项 | 当前还缺什么 |
| 最近更新 | 最近维护时间 |

## 建议维护规则

1. 一个主体一行，不重复建档
2. 执照/地址/联系人变化时同步更新
3. 缺失项必须明确写出，不要空着不管
4. 可用平台要写到平台级，不要只写“可投放”
5. 风险备注记录历史封禁、驳回、限制等
