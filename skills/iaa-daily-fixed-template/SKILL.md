# IAA 固定分析模板

## 一、推荐工作流

每天只做三步：

```text
1. 把数据填进 CSV 模板
2. 跑日报分析脚本
3. 直接看输出结论
```

模板文件：
- `templates/iaa-daily-report-template.csv`

分析脚本：
- `impl/bin/iaa-daily-report-batch.js`

## 三、输出内容

每一天自动输出：

```text
1. 总盘区域判断（利润区/观察区/风险区/止损区）
2. 美加澳判断
3. 日韩判断
4. 核心原因
5. 操作建议
```

---

## 四、适合你现在的执行规则

### 文件修复

```text
- 总盘 D0 >= 100%：可加量
- 美加澳 D0 >= 85%：保守加量
- 日韩 D0 >= 120%：重点加量
```

### 清理

```text
- 总盘 D0 < 65%：优先停量
- H1/H6不达标：不等D0，直接处理
```
