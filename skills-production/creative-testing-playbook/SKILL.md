# 创意测试 SOP

## 测试目标
- 快速识别可放量素材
- 降低低效素材消耗
- 建立稳定的创意迭代机制

## 测试流程
1. 明确测试变量：Hook / 文案 / 画面 / CTA
2. 控制单次只改 1~2 个核心变量
3. 给每组素材设定最小样本量
4. 达到样本量后再做淘汰/晋级判断
5. 胜出素材继续衍生新版本

## 最小样本建议
- 点击量不足：不下结论
- 转化量不足：谨慎判断
- 新素材上线 24h 内：避免过早砍掉

## 淘汰规则示例
- CTR 明显低于组均值
- CPI 明显高于目标
- CTR 和 CVR 同时偏弱
- 疲劳迹象明显

## 晋级规则示例
- CTR 高于组均值
- CPI / CPA 优于目标
- 留存或 ROAS 不差
- 可复制到其他 GEO / 渠道

## 推荐输出
- 本轮测试变量
- 胜出素材
- 淘汰素材
- 下一轮迭代方向
