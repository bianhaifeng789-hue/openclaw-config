# 税务/合规资料准备

## 美国税务表格

### W-8BEN（个人）
- 用于非美国个人收款
- 填写个人信息
- 签名确认

### W-8BEN-E（公司）
- 用于非美国公司收款
- 填写公司信息
- 签名确认

## 欧盟合规

### GDPR要求
- 用户数据保护
- 数据处理透明
- 用户同意机制
- 数据删除权利

### GDPR合规检查清单
```
✅ 隐私政策页面
✅ 数据处理说明
✅ 用户同意弹窗
✅ 数据删除功能
✅ Cookie政策页面
```

## 隐私政策生成

### 模板结构
```
隐私政策

1. 信息收集
   - 收集哪些信息
   - 如何收集信息

2. 信息使用
   - 如何使用信息
   - 信息共享政策

3. 信息保护
   - 数据安全措施
   - 数据存储位置

4. 用户权利
   - 访问权
   - 删除权
   - 更正权

5. Cookie政策
   - Cookie类型
   - Cookie用途

6. 儿童保护（COPPA）
   - 年龄限制
   - 家长同意
```

## COPPA合规

### 要求
- 目标用户年龄限制（13+）
- 儿童数据保护
- 家长同意机制

### 检查清单
```
✅ 年龄验证
✅ 家长同意流程
✅ 儿童数据保护
✅ 隐私政策告知
```
