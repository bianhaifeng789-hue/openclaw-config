# Excel操作指南

运营常用的Excel数据处理技能。

## 常用公式

### 广告收益计算

```
eCPM计算：
=eCPM公式：=(收益/展示数)*1000

示例：
= (B2/C2) * 1000
B2: 收益列
C2: 展示数列

填充率：
= 实际展示/请求次数 * 100%
= D2/E2 * 100

日收益计算：
= DAU * 人均展示 * eCPM / 1000
= F2 * G2 * H2 / 1000
```

### 统计公式

```
求和：
=SUM(B2:B100)

平均值：
=AVERAGE(B2:B100)

最大/最小：
=MAX(B2:B100)
=MIN(B2:B100)

计数：
=COUNT(B2:B100)      数值计数
=COUNTA(B2:B100)     非空计数
```

### 条件公式

```
条件求和：
=SUMIF(条件列, 条件值, 求和列)
示例：=SUMIF(A:A, "AdMob", B:B)
（按平台求收益）

条件计数：
=COUNTIF(条件列, 条件值)
示例：=COUNTIF(C:C, "已批准")
（统计批准数量）

多条件求和：
=SUMIFS(求和列, 条件列1, 条件1, 条件列2, 条件2)
示例：=SUMIFS(B:B, A:A, "AdMob", C:C, "美国")
（按平台和地区求收益）
```

## 数据透视表

### 创建步骤

```
1. 选择数据范围
2. 插入 → 数据透视表
3. 选择放置位置
4. 配置字段：
   - 行：分组维度（如平台）
   - 列：第二维度（如地区）
   - 值：统计指标（如收益求和）
```

### 常用透视表

```
【按平台汇总收益】
行：平台名称
值：收益（求和）

【按日期统计趋势】
行：日期
值：展示数（求和）、收益（求和）

【按地区分析eCPM】
行：地区
值：收益（求和）、展示数（求和）
添加计算字段：eCPM = 收益/展示数*1000
```

## 图表制作

### 收益趋势图

```
步骤：
1. 选择日期和收益列
2. 插入 → 图表 → 折线图
3. 配置：
   - X轴：日期
   - Y轴：收益
4. 添加标题："广告收益趋势"

适用场景：
- 日报/周报趋势展示
```

### 平台对比图

```
步骤：
1. 选择平台和收益数据
2. 插入 → 图表 →柱状图
3. 配置：
   - X轴：平台名称
   - Y轴：收益
4. 添加标题："各平台收益对比"

适用场景：
- 平台效果对比
```

### eCPM分布图

```
步骤：
1. 选择地区和eCPM数据
2. 插入 → 图表 →饼图
3. 配置各地区占比
4. 添加标题："各地区eCPM分布"

适用场景：
- 地区流量分析
```

## 运营专用模板

### APP审核表

```
列设计：
A: APP名称
B: Bundle ID
C: 平台（iOS/Android）
D: 状态（待审核/已批准/已拒绝）
E: 日活
F: eCPM
G: 审核人
H: 审核日期
I: 备注

统计公式：
总APP数：=COUNTA(A:A)
批准数：=COUNTIF(D:D, "已批准")
平均eCPM：=AVERAGE(F:F)
```

### 广告收益日报

```
列设计：
A: 日期
B: APP名称
C: 平台
D: 展示数
E: 收益
F: eCPM（公式：=E2/D2*1000）
G: 填充率
H: 备注

统计公式：
总收益：=SUM(E:E)
平均eCPM：=AVERAGE(F:F)
最高收益日：=MAX(E:E)
```

### 平台注册清单

```
列设计：
A: 平台名称
B: 账户状态
C: 注册日期
D: 负责人
E: APP列表
F: 总收益
G: 备注

筛选：
按状态筛选 → 查看待处理平台
按负责人筛选 → 查看个人负责平台
```

## 数据处理技巧

### 快速填充

```
填充序列：
1. 输入第一个值
2. 拖动填充柄
3. 或双击填充柄（自动填充到末尾）

填充公式：
1. 输入公式到第一个单元格
2. 双击填充柄 → 自动应用到整列
```

### 条件格式

```
高亮异常值：
1. 选择数据列
2. 条件格式 → 突出显示单元格规则
3. 设置条件：
   - 大于/小于某值
   - 重复值
   - 文本包含

示例：
- eCPM > $10 → 绿色高亮
- 填充率 < 80% →红色警示
```

### 数据筛选

```
步骤：
1. 选择标题行
2. 数据 →筛选
3. 点击筛选按钮
4. 选择筛选条件

常用筛选：
- 按平台筛选
- 按状态筛选
- 按日期范围筛选
```

## 快捷键

```
常用操作：
Cmd/Ctrl + C：复制
Cmd/Ctrl + V：粘贴
Cmd/Ctrl + Z：撤销
Cmd/Ctrl + S：保存

选择操作：
Cmd/Ctrl + A：全选
Cmd/Ctrl + Shift + →：选择到行末
Cmd/Ctrl + Shift + ↓：选择到列末

公式操作：
F2：编辑单元格
F4：切换绝对/相对引用
Cmd/Ctrl + ~：显示公式
```
