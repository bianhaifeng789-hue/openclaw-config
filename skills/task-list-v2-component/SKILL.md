# Task List V2 Component

## 功能

展示任务列表。

### 显示内容

- 任务名称
- 状态
- 进度
- 创建时间

### 示例

```
📋 Tasks

1. [Running] 源码分析 - 60%
2. [Pending] 生成报告
3. [Completed] ROI计算 ✅
```
