# AdMob 开通SOP

## 开通入口
```
官网：https://admob.google.com
备用：https://apps.admob.com
```

## 注册要求

### 必备条件
```
✅ Google账号（Gmail）
✅ 公司名称（可填个人名）
✅ 联系邮箱
✅ 联系地址
✅ 银行账户信息（收款）
✅ 税务信息（W-8BEN-E）
```

### 公司资料准备
```json
{
  "companyName": "Your Company LLC",
  "companyAddress": {
    "street": "123 Main Street",
    "city": "San Francisco",
    "state": "CA",
    "zip": "94105",
    "country": "US"
  },
  "contactEmail": "info@company.com",
  "contactPhone": "+1-XXX-XXX-XXXX",
  "website": "https://yourcompany.com"
}
```

### 税务资料准备
```
美国以外公司：W-8BEN-E
美国公司：W-9

填写要点：
- 公司名称必须与注册一致
- 地址必须与注册一致
- 签名必须手写（电子签名也可）
- 有效期3年
```

## 开通流程

### Step 1: 登录Google账号
```
1. 访问 https://admob.google.com
2. 使用Gmail登录
3. 如果没有Gmail，先注册
```

### Step 2: 填写付款信息
```
1. 进入 AdMob 控制台
2. 点击 "付款" → "添加付款方式"
3. 填写银行账户信息：
   - 银行名称
   - 账户号码
   - 路由号码（Routing Number，美国银行）
   - SWIFT代码（非美国银行）
```

### Step 3: 填写税务信息
```
1. 点击 "付款" → "税务信息"
2. 选择 "美国以外实体" → W-8BEN-E
3. 填写表单：
   - 公司名称
   - 公司地址
   - 税务居住国
   - 签名
4. 提交表单
```

### Step 4: 创建应用
```
1. 点击 "应用" → "添加应用"
2. 选择平台：Android / iOS
3. 填写应用名称
4. 是否已在商店？选择 "是" 或 "否"
5. 如果 "是"，填写商店链接
6. 如果 "否"，手动填写包名
```

### Step 5: 创建广告位
```
1. 进入应用详情
2. 点击 "广告单元" → "创建广告单元"
3. 选择广告类型：
   - Banner（横幅）
   - Interstitial（插屏）
   - Rewarded（激励视频）
   - Native（原生）
4. 填写广告单元名称
5. 点击 "创建"
6. 记录广告单元ID（ca-app-pub-XXXXX/XXXXX）
```

## 审核流程

### 自动审核
```
1. 提交应用后，AdMob自动审核
2. 审核时间：1-7天
3. 审核内容：
   - 应用合规性
   - 内容是否符合政策
   - 是否有足够内容
```

### 审核状态
```
- 待审核：黄色标记
- 审核通过：绿色标记
- 审核拒绝：红色标记（需修改）
```

## 常见被拒原因

### 1. 应用内容不足
```
问题：应用内容太少，只是空壳
解决：添加足够内容（至少10个页面/关卡）
```

### 2. 违反政策
```
问题：应用包含违规内容（成人、暴力、赌博等）
解决：移除违规内容，重新提交
```

### 3. 版权问题
```
问题：应用使用他人版权素材
解决：使用原创素材，或获得授权
```

### 4. 应用未上架
```
问题：应用未上架商店，但填了 "已在商店"
解决：先上架商店，再提交审核
```

### 5. 广告位违规
```
问题：广告位位置违规（如诱导点击）
解决：调整广告位位置，符合政策
```

## 开通后第一步配置

### 1. 验证收款信息
```
- 检查银行账户是否正确
- 检查税务信息是否提交
- 等待验证邮件
```

### 2. 设置广告位
```
- 创建至少3个广告位：Banner + 插屏 + 激励
- 记录广告单元ID
- 在应用中集成SDK
```

### 3. 测试广告
```
- 使用测试设备ID测试广告
- 确认广告正常展示
- 测试不同广告类型
```

### 4. 配置中介
```
- 进入 "中介" 页面
- 创建中介组
- 添加其他广告网络
- 设置eCPM floor（可选）
```

## 注意事项

### 防封号要点
```
❌ 不要诱导点击广告
❌ 不要自己点击广告
❌ 不要展示过多广告（频控）
❌ 不要在违规内容旁展示广告
❌ 不要隐藏广告内容
❌ 不要使用虚假流量

✅ 合理设置广告频控（每小时≤3次插屏）
✅ 广告位置明显可见
✅ 用户主动触发激励视频
✅ 遵守AdMob政策
```

### 新号建议
```
1. 前7天展示量控制在5000以下
2. 不要频繁修改广告位
3. 不要频繁切换收款信息
4. 保持稳定的展示量增长
5. 如被拒，仔细阅读原因再申诉
```
