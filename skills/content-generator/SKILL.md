# 英语内容生成技能

专为美国市场设计的英语内容生成工具。

## 广告文案模板

### Facebook 广告

**模板 1：问题-解决型**

```
Headline: {痛点}?

Body: 
✅ {产品名} - {核心卖点}
• {benefit_1}
• {benefit_2}
• {benefit_3}

👉 {CTA}: {链接}

示例：
---
Headline: Struggling to track your ad ROI?

Body:
✅ AdTrack Pro - Real-time analytics dashboard
• Monitor all campaigns in one place
• ROI calculator included
• Works with Facebook, Google, TikTok

👉 Try Free: [link]
---
```

**模板 2：优惠促销型**

```
Headline: 🎉 {折扣} OFF - Limited Time!

Body:
{产品名} - {一句话描述}

⏰ Offer ends: {日期}

Use code: {优惠码}

Shop now: {链接}

示例：
---
Headline: 🎉 30% OFF - Limited Time!

Body:
AdTrack Pro - The #1 analytics tool for US marketers

⏰ Offer ends: Dec 31

Use code: NEWYEAR30

Shop now: [link]
---
```

### Google Ads

**模板：搜索广告**

```
Headline 1: {核心关键词} | {品牌}
Headline 2: {卖点} | {优惠}
Headline 3: {CTA} Today

Description 1: {详细卖点} {数字证明}
Description 2: {优惠信息} {CTA}

示例：
---
Headline 1: Ad Analytics Tool | AdTrack
Headline 2: Track ROI in Real-Time | Free Trial
Headline 3: Start Today

Description 1: Monitor campaigns across platforms. Used by 10,000+ marketers.
Description 2: 30-day free trial. No credit card required. Sign up now.
---
```

**字符限制：**
- Headline: 30 字符
- Description: 90 字符

### TikTok 广告

**模板：短视频脚本**

```
【0-3秒】Hook: 
{吸引注意力的开场}

【3-15秒】Problem:
{展示痛点}

【15-30秒】Solution:
{产品演示}

【30-35秒】CTA:
{行动号召}

示例：
---
【0-3秒】Hook:
"POV: You're spending $1000 on ads..."

【3-15秒】Problem:
"But can't tell which one actually works 😭"

【15-30秒】Solution:
"AdTrack shows your ROI in REAL TIME 📊"
[展示界面操作]

【30-35秒】CTA:
"Link in bio - Free trial! 👇"
---
```

## A/B 测试文案变体

```

```

```
A: "Try Free"

C: "Learn More"
D: "Sign Up Now"
```

## 写作风格指南（美国市场）

| 要点 | 建议 |
|------|------|
| **语气** | 热情但专业 |
| **语言** | 简洁，避免复杂句 |
| **数字** | 用具体数字增加可信度 |
| **Emoji** | 适度使用（社交媒体） |
| **CTA** | 明确、紧迫感 |
| **排版** | 多用换行，易于阅读 |
