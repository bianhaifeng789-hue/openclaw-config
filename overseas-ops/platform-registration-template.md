# 平台注册资料模板（第一版）

> 用途：沉淀不同海外平台/广告平台/中介平台的开户资料要求，并和主体资料做映射。

## 一、平台资料字段模板

| 字段 | 说明 | 来源 |
|---|---|---|
| 平台名称 | 目标平台名称 | 平台要求 |
| 平台类型 | 广告平台 / 中介平台 / 商店 / 商务平台 | 平台要求 |
| 开户地区 | 面向哪个地区开户 | 平台要求 |
| 需要主体国家 | 平台要求的主体地区 | 平台要求 |
| 公司英文名 | 公司注册英文名 | 主体资料 |
| 公司中文名 | 公司中文名 | 主体资料 |
| 注册号 | 公司注册号 | 主体资料 |
| 执照/注册证书 | 上传文件 | 主体资料 |
| 地址证明 | 是否需要 | 主体资料/补充 |
| 税务信息 | VAT/EIN/税号等 | 主体资料 |
| 联系人姓名 | 对接人 | 主体资料 |
| 联系邮箱 | 注册邮箱 | 主体资料 |
| 联系电话 | 注册电话 | 主体资料 |
| 网站/App 信息 | 产品链接、官网、商店页 | 产品资料 |
| 收款方式 | 银行账户/第三方收款 | 财务资料 |
| 审核备注 | 平台特殊要求 | 平台经验 |
| 缺失项 | 当前未准备好的材料 | 检查结果 |
| 状态 | 待准备 / 可提交 / 已提交 / 被驳回 | 进度状态 |

## 三、主体-平台映射建议

- 一个平台注册前，先确认主体是否满足地区要求
- 主体资料优先复用，平台差异字段单独补
- 被驳回的平台要记录驳回原因，避免重复踩坑
- 缺失项统一汇总，方便后续一次补齐
