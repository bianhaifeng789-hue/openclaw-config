# 飞书云文档操作指南

运营常用的飞书文档操作。

## 创建文档

### 创建新文档

```
步骤：
1. 打开飞书 → 云文档
2. 点击 "新建" → "文档"
3. 输入文档标题
4. 开始编辑

快捷键：
- Cmd/Ctrl + B：加粗
- Cmd/Ctrl + I：斜体
- Cmd/Ctrl + U：下划线
- Cmd/Ctrl + K：插入链接
```

### 文档格式

```
支持的格式：
- 标题（H1-H6）
- 正文
- 列表（有序/无序）
- 任务列表
- 引用
- 代码块
- 表格
- 图片
- 链接
- 分割线
```

---

## 表格操作

### 创建表格

```
步骤：
1. 文档内点击 "+"
2. 选择 "表格"
3. 设置行数列数
4. 开始填写

最大：
- 100行
- 20列
```

### 表格功能

```
功能：
✅ 单元格合并
✅ 行列插入/删除
✅ 单元格颜色
✅ 文字格式
✅ 对齐方式
✅ 排序（部分）
```

---

## 多维表格

### 创建多维表格

```
步骤：
1. 云文档 → "新建" → "多维表格"
2. 选择模板或空白
3. 配置字段类型
4. 开始添加数据

字段类型：
- 文本
- 数字
- 单选
- 多选
- 日期
- 人员
- 图片
- 附件
- URL
- 公式
```

### 常用视图

```
视图类型：
- 表格视图：传统表格
- 看板视图：任务管理
- 日历视图：日程安排
- 甘特图：项目进度
```

## 常用场景模板

### APP审核表格

```
字段设计：
- APP名称（文本）
- Bundle ID（文本）
- 平台（单选：iOS/Android）
- 状态（单选：待审核/已批准/已拒绝）
- 日活（数字）
- eCPM（数字）
- 审核人（人员）
- 审核日期（日期）
- 备注（文本）
```

### 广告收益统计

```
字段设计：
- 日期（日期）
- APP名称（文本）
- 平台（单选）
- 展示数（数字）
- 收益（数字）
- eCPM（公式：收益/展示数*1000）
- 填充率（数字）
- 备注（文本）
```

### 平台注册清单

```
字段设计：
- 平台名称（单选：AdMob/Meta/AppLovin等）
- 账户状态（单选）
- 注册日期（日期）
- 负责人（人员）
- APP列表（多选）
- 收益（数字）
- 备注（文本）
```

## 文档协作

### 分享文档

```
步骤：
1. 点击右上角 "分享"
2. 设置权限：
   - 可编辑
   - 可查看
   - 仅评论
3. 添加协作者
4. 或获取分享链接

权限级别：
- 所有者：完全控制
- 可编辑：可修改内容
- 可评论：可添加评论
- 可查看：仅阅读
```

### 评论功能

```
使用：
1. 选中文本
2. 点击 "评论"
3. 输入评论内容
4. @相关人员

用途：
- 审核意见
- 问题反馈
- 协作讨论
```

## API集成（可选）

### feishu_doc工具

```
可用操作：
- read：读取文档内容
- write：写入文档内容
- append：追加内容
- create：创建新文档
- list_blocks：列出文档块
- update_block：更新文档块

示例：
读取文档：
feishu_doc(action="read", doc_token="xxx")
```

### feishu_bitable工具

```
可用操作：
- get_meta：获取表格信息
- list_records：列出记录
- create_record：创建记录
- update_record：更新记录

示例：
创建记录：
feishu_bitable_create_record(
  app_token="xxx",
  table_id="yyy",
  fields={"APP名称": "TestApp", "状态": "待审核"}
)
```
