# 用户个人主页 - UI设计规范

## 📐 页面结构布局

采用**单列滚动式布局**，自上而下分为：

```
┌─────────────────────────────────┐
│  ← 返回              设置 ⚙️    │ ← 导航栏
├─────────────────────────────────┤
│         ┌──────────┐            │
│         │  头像    │ (80pt)     │ ← 用户信息区
│         └──────────┘            │
│          小明同学               │ (昵称)
│     热爱生活，喜欢旅行 📷       │ (简介)
│                                 │
│   1234 粉丝 | 567 关注 | 89 动态 │ ← 统计区
│                                 │
│  [ 编辑资料 ]    [ 分享 ]       │ ← 操作按钮
├─────────────────────────────────┤
│    动态  |  喜欢                │ ← 标签栏
├─────────────────────────────────┤
│  ┌───────────────────────────┐  │
│  │ 今天去了一趟西湖...       │  │ ← 动态卡片
│  │ [图片]                    │  │
│  │ 2小时前        ❤️ 45      │  │
│  └───────────────────────────┘  │
│  ┌───────────────────────────┐  │
│  │ 分享一张日落照片...       │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

## 🎨 色彩规范

| 元素 | 颜色值 | 用途 |
|------|--------|------|
| **主色** | `#FF6B9D` | 按钮高亮、关注状态、点赞数 |
| **背景色** | `#F5F5F7` | 页面背景、区分卡片与背景 |
| **卡片背景** | `#FFFFFF` | 动态卡片、信息区背景 |
| **主文字** | `#1A1A1A` | 昵称、动态内容 |
| **次要文字** | `#8E8E93` | 简介、统计标签、时间 |
| **分隔线** | `#E5E5EA` | 0.5pt细线分隔 |
| **边框** | `#E5E5EA` | 编辑框边框 |

## 📏 尺寸规范

### 头像
- 尺寸: 80pt × 80pt
- 圆角: 40pt (圆形)
- 外边距: 12pt (底部)

### 按钮
- 主按钮宽度: 自适应文字 + 48pt padding
- 按钮高度: 40pt
- 圆角: 8pt
- 间距: 12pt (按钮间)

### 卡片
- 卡片内边距: 16pt
- 卡片间距: 8pt
- 图片尺寸: 100pt × 100pt
- 图片圆角: 8pt

---

## 👆 关键交互点

### 1. 头像点击
- **触发**: 点击头像区域
- **效果**: 弹出大图预览（支持缩放）
- **动画**: 渐显 + 缩放过渡

### 2. 简介编辑
- **触发**: 点击简介文字
- **效果**: 弹出编辑框（TextInput）
- **限制**: 200字上限
- **保存**: 点击"保存"按钮提交

### 3. 粉丝/关注跳转
- **触发**: 点击粉丝/关注数字
- **效果**: 跳转到关系列表页
- **动画**: 右滑推入

### 4. 下拉刷新
- **触发**: 页面顶部下拉
- **效果**: 显示刷新指示器，重新加载数据
- **动画**: 标准下拉刷新动画

### 5. 动态卡片左滑
- **触发**: 卡片左滑超过 100pt
- **效果**: 显示"删除"/"置顶"按钮
- **动画**: 水平滑动 + 按钮渐显

## 📱 平台适配

### iOS
- 遵循 iOS Human Interface Guidelines
- 使用 SF Pro 系统字体
- 导航栏高度: 44pt
- 底部安全区: 34pt

### Android
- 遵循 Material Design 3
- 使用 Roboto 字体
- 导航栏高度: 56dp
- 底部安全区: 24dp
