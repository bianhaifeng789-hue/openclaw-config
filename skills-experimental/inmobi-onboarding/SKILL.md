# InMobi 开通SOP

## 开通入口
```
官网：https://www.inmobi.com
开发者门户：https://www.inmobi.com/publisher
```

## 注册要求
```
✅ 公司名称
✅ 联系邮箱
✅ 应用信息（至少1个）
✅ 银行账户信息
✅ 税务信息
```

## 开通流程
```
Step 1: 注册账号
- 访问 https://www.inmobi.com/publisher
- 点击 "Sign Up"
- 填写公司信息
- 提交注册

Step 2: 创建应用
- 进入 Dashboard
- 点击 "Add Property"
- 选择平台：Android / iOS
- 填写应用信息

Step 3: 创建广告位
- 选择广告类型：
  * Banner
  * Interstitial
  * Rewarded Video
  * Native
- 记录 Placement ID

Step 4: 配置收款
- Account → Payment Settings
- 填写银行账户信息
- 填写税务信息
```

## 特点
```
✅ 印度系平台，全球覆盖
✅ Banner表现优秀
✅ 非游戏类友好
✅ 美国量中等
```

## SDK集成
```
Android:
implementation 'com.inmobi.monetization:inmobi-sdk:+'

iOS:
pod 'InMobiSDK'
```
