# Mintegral 开通SOP

## 开通入口
```
官网：https://www.mintegral.com
开发者门户：https://dashboard.mintegral.com
```

## 注册要求
```
✅ 公司名称
✅ 联系邮箱
✅ 应用信息（至少1个）
✅ 银行账户信息
✅ 税务信息
```

## 开通流程
```
Step 1: 注册账号
- 访问 https://dashboard.mintegral.com
- 点击 "Sign Up"
- 填写公司信息
- 提交注册

Step 2: 创建应用
- 进入 Dashboard
- 点击 "Add App"
- 选择平台：Android / iOS
- 填写应用信息

Step 3: 创建广告位
- 选择广告类型：
  * Interstitial
  * Rewarded Video
  * Banner
- 记录 App ID 和 Placement ID

Step 4: 配置收款
- Account → Payment Settings
- 填写银行账户信息
```

## 特点
```
✅ 中国系平台
✅ 亚洲市场表现好
✅ 海外量较少
⚠️ 审核较严格（14-30天）
```

## SDK集成
```
Android:
implementation 'com.mintegral:mintegral-sdk:+'

iOS:
pod 'MintegralAdSDK'
```
