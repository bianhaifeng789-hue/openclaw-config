## LTV核心概念

### 什么是LTV？

```
LTV（Life Time Value）= 用户生命周期总价值

广告变现LTV：
用户从安装到流失期间产生的总广告收益

公式：
LTV = Σ（每日ARPU × 当日留存率）
```

---

## 广告变现LTV计算

### 方法一：简单估算法

```
LTV = ARPU × 平均生命周期天数

示例：
- ARPU：$0.10/天
- 平均生命周期：30天
- LTV = 0.10 × 30 = $3.00

适用场景：
- 快速估算
- 产品初期阶段
```

### 方法二：留存率累计法

```
LTV = Σ（ARPU_d × Retention_d）

d = 第1天, 第2天, ... 第N天

示例：
Day 1: ARPU $0.15 × Retention 100% = $0.15
Day 7: ARPU $0.12 × Retention 40% = $0.048
Day 30: ARPU $0.08 × Retention 15% = $0.012
...
累计 = LTV

计算表格：
| 天数 | ARPU | 留存率 | 收益贡献 |
|------|------|--------|----------|
| D1 | $0.15 | 100% | $0.15 |
| D7 | $0.12 | 40% | $0.048 |
| D14 | $0.10 | 25% | $0.025 |
| D30 | $0.08 | 15% | $0.012 |
| D60 | $0.06 | 8% | $0.0048 |
| D90 | $0.05 | 5% | $0.0025 |
| **累计LTV** | | | **$0.24+** |
```

### 方法三：衰减模型法

```
LTV = ARPU × (1 / (1 - 留存衰减率))

留存衰减率计算：
r = (Retention_D7 / Retention_D1)^(1/7)

示例：
- D1留存：50%
- D7留存：20%
- 衰减率 r = (0.20/0.50)^(1/7) = 0.91

LTV = 0.10 × (1 / (1 - 0.91)) = $1.11

适用场景：
- 长期预测
- 稳定产品
```

## 关键指标

### ARPU

```
ARPU（Average Revenue Per User）= 平均每用户收益

公式：
ARPU = 日总收益 / 日活跃用户数

示例：
- 日收益：$500
- DAU：5000
- ARPU = 500 / 5000 = $0.10

行业基准（美国广告变现游戏）：
| 类型 | ARPU范围 |
|------|----------|
| 休闲游戏 | $0.05-0.15 |
| 中重度游戏 | $0.10-0.30 |
| 超休闲游戏 | $0.02-0.08 |
```

### ARPPU

```
ARPPU（Average Revenue Per Paying User）= 平均每付费用户收益

（广告变现产品通常不适用，除非有内购）
```

### 留存率

```
留存率 = 当日活跃用户 / 安装日用户数

关键留存节点：
- D1留存：40-60%（正常）
- D7留存：15-30%（正常）
- D30留存：5-15%（正常）

行业基准（休闲游戏美国市场）：
| 指标 | 优秀 | 正常 | 较差 |
|------|------|------|------|
| D1 | >60% | 40-60% | <40% |
| D7 | >30% | 15-30% | <15% |
| D30 | >15% | 5-15% | <5% |
```

## 买量ROI评估

### CPI vs LTV

```
判断买量是否盈利：

ROI = (LTV - CPI) / CPI × 100%

示例：
- CPI（单用户买量成本）：$0.40
- LTV：$0.65
- ROI = (0.65 - 0.40) / 0.40 = 62.5% ✅ 盈利

盈亏平衡点：
CPI = LTV 时，ROI = 0

建议：
- CPI < LTV × 0.7（安全区间）
- CPI < LTV × 0.8（正常区间）
- CPI > LTV（亏损，不推荐）
```

### 回收周期

```
回收周期 = CPI / 日ARPU

示例：
- CPI：$0.40
- ARPU：$0.12
- 回收周期 = 0.40 / 0.12 = 3.3天

建议：
- < 7天：优秀
- 7-14天：正常
- > 14天：风险较高
```

## LTV预测模板

### Excel计算公式

```
列设计：
A: 天数（1, 2, 3, ... 90）
B: 留存率（实测/预测）
C: ARPU
D: 收益贡献 = B × C
E: 累计LTV = SUM(D$2:D2)

公式示例：
D2: =B2*C2
E2: =SUM($D$2:D2)
E90: =SUM($D$2:$D$90) → 最终LTV
```

### 快速估算表

```

= ARPU × (D1留存 × 1 + D7留存 × 6)

【D30 LTV估算】
= ARPU × (D1 × 1 + D7 × 6 + D30 × 23)

= ARPU × (D1 × 1 + D7 × 6 + D30 × 23 + D90 × 60)
```
