# 留存分析

## 留存率计算

### 日留存
```
Day 1留存 = Day 1活跃用户 / 新增用户
Day 7留存 = Day 7活跃用户 / 新增用户
Day 30留存 = Day 30活跃用户 / 新增用户
```

### 周留存
```
Week 1留存 = Week 1活跃用户 / 新增用户
Week 4留存 = Week 4活跃用户 / 新增用户
Week 12留存 = Week 12活跃用户 / 新增用户
```

### 月留存
```
Month 1留存 = Month 1活跃用户 / 新增用户
Month 3留存 = Month 3活跃用户 / 新增用户
Month 6留存 = Month 6活跃用户 / 新增用户
```

## 留存曲线模型

### 幂律模型
```
留存率 = a * (day ^ b)
```

### 指数模型
```
留存率 = a * exp(-b * day)
```

## 留存影响因素

- 游戏质量（核心玩法）
- 新手引导（首次体验）
- 推送策略（召回机制）
- 社交功能（社区粘性）
- 更新频率（内容新鲜度）

## 留存对比分析

### 按渠道对比
```
渠道A: Day 7留存 = 15%
渠道B: Day 7留存 = 12%
渠道C: Day 7留存 = 18%
结论: 渠道C留存最高，优先投放
```

### 按版本对比
```
版本1.0: Day 7留存 = 10%
版本2.0: Day 7留存 = 15%
版本3.0: Day 7留存 = 20%
结论: 版本更新提升留存
```
