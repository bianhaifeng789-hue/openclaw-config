# 账号隔离管理

## 防关联策略

### 1. IP隔离
- 使用不同代理IP登录不同账号
- 推荐工具：Luminati, Smartproxy
- 避免同一IP登录多个账号

### 2. 设备隔离
- 使用不同浏览器指纹
- 推荐工具：Multilogin, GoLogin
- 独立浏览器容器

### 3. 资料隔离
- 每个账号使用不同的：
  - 公司名称
  - 联系邮箱
  - 联系电话
  - 地址信息

### 4. Cookie隔离
- 独立浏览器会话
- 定期清理Cookie
- 避免跨账号Cookie共享

## 账号健康检测

### 检测项目
- 账号状态（活跃/受限/封禁）
- 收款状态（正常/异常）
- 广告位状态（正常/暂停）
- 收入趋势（正常/异常）

### 异常告警
- 收入突然下降
- 账号受限通知
- 广告位暂停
- 付款延迟

## 账号切换流程

1. 检查账号状态
2. 切换浏览器容器
3. 切换IP地址
4. 登录目标账号
5. 记录切换日志
