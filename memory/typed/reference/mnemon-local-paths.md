Mnemon 当前本地关键路径：

- Binary: `~/.local/bin/mnemon`
- Data dir: `~/.mnemon/`
- DB: `~/.mnemon/data/default/mnemon.db`
- Prompt guide: `~/.mnemon/prompt/guide.md`
- Prompt skill: `~/.mnemon/prompt/skill.md`
- OpenClaw skill: `~/.openclaw/skills/mnemon/SKILL.md`
- OpenClaw hook: `~/.openclaw/hooks/mnemon-prime/`
- OpenClaw extension: `~/.openclaw/extensions/mnemon/`
