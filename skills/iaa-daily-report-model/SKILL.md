# IAA 日报分析模型

## 一、适用场景

适合你的两类低留存短回收产品：
- 文件修复
- 清理工具

这类产品不看长线LTV，而看：

```text
H1 / H6 / H12 / D0 / CPI / 分地区结构
```

---

## 二、日报必须包含字段

### 基础字段

```text
日期
渠道花费
渠道安装
渠道CPI
首日回收
首日ROI
1小时ROI
6小时ROI
12小时ROI
```

### 地区字段

```text

美加澳CPI

美加澳首日ROI
日韩1小时ROI
日韩6小时ROI
日韩12小时ROI
日韩首日ROI
```

## 三、产品阈值

### 文件修复

```text
总体阈值：
H1 >= 22%
H6 >= 58%
H12 >= 73%
D0 >= 85%

美加澳：
H1 >= 40%
H6 >= 50%
H12 >= 52%
D0 >= 85%

日韩：
H1 >= 60%
H6 >= 65%
H12 >= 70%
D0 >= 100%
```

### 清理

```text
总体阈值：
H1 >= 15%
H6 >= 42%
H12 >= 55%
D0 >= 65%

美加澳：
H1 >= 30%
H6 >= 40%
H12 >= 48%
D0 >= 70%

日韩：
H1 >= 45%
H6 >= 55%
H12 >= 60%
D0 >= 85%
```

---

## 四、决策区间

### 总盘

```text
D0 >= 100% -> 利润区，可放量
D0 85%~100% -> 观察区，可保量/小幅加量
D0 75%~85% -> 风险区，谨慎
D0 < 75% -> 止损区，优先停量/换素材
```

### 地区

#### 美加澳

```text
D0 >= 85% -> 可投
D0 75%~85% -> 保守观察
D0 < 75% -> 不建议继续放量
```

#### 日韩

```text
D0 >= 120% -> 强利润区，可重点加量
D0 100%~120% -> 可投
D0 85%~100% -> 观察
D0 < 85% -> 谨慎或停量
```

---

## 五、日报输出结构

每次分析输出四部分：

### 1. 总结论

```text
- 今日整体属于：利润区 / 观察区 / 止损区
- 核心原因：CPI变化 / 地区结构 / 小时回收
```

### 2. 分地区结论

```text
- 美加澳：拖后腿 / 稳定 / 可放量
- 日韩：主利润区 / 观察 / 不达标
```

### 3. 小时级判断

```text
- H1 看创意质量
- H6 看是否还能继续
- H12 看D0是否定型
```

### 4. 操作建议

```text
- 加量
- 保量
- 降量
- 停投
- 换素材
- 压CPI
```

---

## 六、你当前业务的执行规则

### 文件修复

```text
若 美加澳 D0 >= 85% 且 H6 >= 50% -> 保守加量
若 日韩 D0 >= 120% 且 H6 >= 65% -> 优先加量
若 总盘 D0 < 85% -> 不要扩量
```

### 清理

```text
若 D0 < 65% -> 基本停
若 H3/H6 不达标 -> 不等D0，直接处理
核心目标永远是：压CPI + 拉高首小时价值
```

---

## 七、一句话总结

```text
日报不是看总ROI一个数，而是看：
总盘能不能活，美加澳是不是拖后腿，日韩是不是利润池。
```
