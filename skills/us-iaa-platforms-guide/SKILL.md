## 各平台优缺点对比

### AdMob
```
优点：
✅ Google生态，审核相对宽松
✅ 美国eCPM最高（$8-15）
✅ 所有中介都支持
✅ Banner/插屏/激励全覆盖
✅ 文档完整，社区成熟

缺点：
❌ 新号需要一定的展示量才能进入竞价池
❌ 收益波动较大（受Google算法影响）
❌ 封号风险（需注意合规）
```

### AppLovin MAX
```
优点：
✅ 开通极快（1-2天）
✅ Bidding完整，竞价透明
✅ 美国表现优秀（eCPM $10-20）
✅ 自家中介MAX成熟
✅ SDK集成简单

缺点：
❌ 只能在MAX中介中使用（不能接入其他中介）
❌ 需要使用MAX SDK
```

### Meta Audience Network
```
优点：
✅ Facebook流量池，量大
✅ 美国eCPM高（$8-12）
✅ Bidding完整
✅ 所有中介都支持

缺点：
❌ 审核严格（需FB开发者账号）
❌ 封号风险高（政策敏感）
❌ 需要绑定Facebook账号
❌ 新号需要一定的展示量
```

### Unity Ads
```
优点：
✅ 游戏开发者友好
✅ 激励视频表现好
✅ SDK集成简单
✅ 美国eCPM中等（$6-10）
✅ 所有中介都支持

缺点：
❌ Banner效果一般
❌ 非游戏类效果较差
```

### ironSource
```
优点：
✅ LevelPlay中介成熟
✅ 开通快
✅ 美国市场覆盖好
✅ 激励视频表现优秀
✅ Bidding完整

缺点：
❌ 只能在LevelPlay中介中使用
❌ 需要使用ironSource SDK
```
