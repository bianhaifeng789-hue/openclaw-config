# Edit Article

## Process

### 1. Divide into sections

Divide article by headings. Think about main points per section.

**Information is DAG** - pieces depend on other pieces. Ensure section order respects dependencies.

Confirm sections with user.

### 2. Rewrite each section

Improve clarity, coherence, flow.

**Rule: Maximum 240 characters per paragraph**

## Best Practices

- Information as directed acyclic graph
- Respect dependency order
- Tight prose (240 chars/paragraph)
- Clear section structure
